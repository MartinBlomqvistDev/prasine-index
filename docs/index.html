Architecture

Seven agents. One evidence chain.

Each agent has a single responsibility. Data flows through typed Pydantic models at every boundary — no raw strings, no untyped dicts between steps.

1
Discovery Agent
Monitors EU company IR pages, press releases, and CSRD reports for new content via SHA-256 change detection.
2
Extraction Agent
Extracts every green claim verbatim with forced tool use. Returns structured List[Claim].
Raw Anthropic SDK
3
Context Agent
Queries PostgreSQL for historical claims, aggregate scores, and semantically similar prior claims via pgvector.
PostgreSQL + pgvector
4
Verification Agent
Queries 22 open data sources in parallel — including the claim's own source document, fetched and analysed via the Anthropic document API. Partial failures are tolerated.
LangGraph
5
Lobbying Agent
Cross-references the company against the EU Transparency Register. A company lobbying against climate legislation while claiming green leadership is flagged explicitly.
6
Judge Agent
LLM-as-judge. Produces a calibrated 0–100 score with per-dimension breakdown and full chain-of-thought reasoning preserved verbatim for audit.
Raw Anthropic SDK
7
Report Agent
Generates a publication-ready Markdown report. Every factual assertion is cited. Every data gap is disclosed.
Raw Anthropic SDK
Why LangGraph only at step 4? The Verification Agent queries 22 independent data sources concurrently and must handle partial failures gracefully — exactly the fan-out / partial-state-merge problem LangGraph's StateGraph with operator.add reducers solves. The other six agents are single-step operations where raw SDK gives better prompt control and cleaner code. This is when to use a framework, not just that frameworks exist.

Data Sources

22 open data sources.

All EU open data. No proprietary APIs. Every source is independently authoritative. The 22nd source is the document the claim itself cites — fetched and analysed via the Anthropic document API.

EU ETS / EUTL
Verified annual CO₂ emissions per installation, 2005–present. Third-party verified under EU Regulation 601/2012.
Active — daily snapshot
SBTi
Science-based targets — validated, committed, and removed. A removed target is a CONFIRMED_GREENWASHING signal at 0.95 confidence.
Active — quarterly
LobbyMap
Corporate climate policy engagement scores A+ to F. A D/E/F company claiming climate leadership is textbook greenwashing.
Active — bulk CSV
Enforcement Rulings
ASA, ACM, AGCM, CMA, EC rulings and court judgments. A prior ruling is the strongest possible evidence — an authority has already decided.
Active — curated static
CA100+
Net-zero benchmark from 700+ investors ($68tn AUM) for the 170 largest emitters. "Not Aligned" contradicts net-zero claims.
Active — annual
E-PRTR
EEA-verified non-CO₂ GHG releases (CH₄, N₂O, HFCs) per facility — emissions not covered by EU ETS.
Active — annual
Banking on Climate Chaos
Fossil fuel financing by the 60 largest banks, 2016–present. Banks claiming net zero while financing hundreds of billions in fossil expansion.
Active — annual
Global Coal Exit List
~1,000 companies active in the coal value chain. Expanding coal capacity while claiming clean energy transition.
Active — annual
EU Transparency Register
Brussels lobbying declarations. Companies lobbying against climate legislation while claiming green leadership is flagged explicitly.
Active — XLSX bulk export
EEA National Totals
EEA-verified national GHG inventory totals per country, 1990-present. Official EU submission benchmarks for company percentage claims.
Active — annual
EU Innovation Fund
